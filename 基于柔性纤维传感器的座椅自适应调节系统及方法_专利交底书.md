# 专利交底书：基于柔性纤维传感器的座椅自适应调节系统及方法

## 1. 发明名称
基于柔性纤维传感器的座椅自适应调节系统及方法

## 2. 所属技术领域
本发明涉及汽车座椅智能控制技术领域，具体涉及一种基于柔性纤维体压传感器的汽车座椅自适应调节系统及方法。

## 3. 背景技术
### 3.1 现有技术情况
汽车座椅的舒适性是驾乘体验与健康的核心指标。体压分布情况是衡量人体乘坐舒适性最主要的生物力学因素。目前市场上的自适应座椅调节系统多采用薄膜压力传感器或摄像头视觉识别技术。
薄膜压力传感器存在以下问题：
1. 柔韧性差，难以完美贴合座椅复杂的曲面，容易产生褶皱，影响乘坐舒适感；
2. 耐久性不足，在长期使用和弯折后容易出现信号漂移、灵敏度下降；
3. 透气性差，影响座椅的通风和加热功能。

摄像头视觉识别技术存在以下问题：
1. 只能识别乘员的大致体型轮廓，无法获取真实的受力分布数据；
2. 容易受光线、衣物遮挡等环境因素影响，识别准确率不稳定；
3. 存在侵犯乘员隐私的风险。

### 3.2 现有技术的缺点
现有座椅自适应调节方案无法兼顾高精度体压感知、高乘坐舒适性、长使用寿命以及精准的动态自适应调节。缺乏一种能够实时、精准感知体压分布，且不影响座椅本身舒适性，并能根据体压数据进行多区域气囊协同调节的闭环系统。

## 4. 发明内容
### 4.1 发明目的
本发明的目的在于解决现有技术中传感器舒适性差、耐久度低、体型识别不准、气囊调节缺乏闭环反馈等问题，提供一种基于柔性纤维传感器的座椅自适应调节系统及方法。

### 4.2 技术方案
本发明提供了一种基于柔性纤维传感器的座椅自适应调节系统，其特征在于，包括：
**柔性纤维体压传感器阵列**：嵌入于汽车座椅的座垫和靠背区域，用于实时采集乘员与座椅接触界面的体压分布数据。该传感器采用织物结构的本征柔性材料，传感器节点通过横纵交叉排布形成，最小折叠半径小于0.2mm，采样频率不低于13Hz。
**数据采集与处理模块**：与传感器阵列连接，用于接收并处理体压分布数据。
**主控单元（算法模块）**：内置座椅自适应调节算法、乘员体型自识别算法和品味系数算法。根据处理后的体压分布数据识别乘员体型，并输出目标气囊状态。
**气囊执行机构**：包括分布于座椅肩部、腰托、侧翼、臀部、腿部等多个区域的支撑气囊，根据主控单元的指令进行充放气调节。
**可视化交互终端**：车机端安卓软件，用于实时显示3D体压分布云图、气囊调节状态，并提供用户自定义气囊调节界面。

本发明还提供了一种基于柔性纤维传感器的座椅自适应调节方法，包括以下步骤：
1. **落座检测与数据采集**：乘员落座后，柔性纤维体压传感器阵列实时扫描并采集乘员与座垫、靠背的体压分布数据；
2. **体型自识别**：主控单元基于采集到的体压分布特征，通过体型三分类算法（瘦小体型、中等体型、高大体型）识别乘员体型；
3. **匹配与调节**：主控单元查询本地数据库，判断该体型是否存在自定义的“品味系数”（气囊偏好设置）。若存在，则调用该参数作为基准进行调节；若不存在，则根据体型识别结果输出默认的舒适气囊状态指令；
4. **动态闭环控制**：在行车过程中，传感器阵列持续获取实时体压数据，主控单元将其与目标体压分布阈值进行对比。当体压分布超出正常阈值范围时，自动控制对应区域的气囊进行充放气微调，使体压分布动态保持在预设舒适区间内。

### 4.3 关键技术点（发明点）
1. **新型传感材质与集成方式**：采用可Z型死折、耐揉搓的柔性纤维传感器，不改变座椅原有的透气性和柔软度，实现传感器与座椅面套的无缝模块化集成。
2. **基于体压分布的体型三分类算法**：摒弃了传统的摄像头视觉识别，仅通过体压分布的比例和特征值模型，即可准确将乘员划分为瘦小（<50kg）、中等（55~80kg）、高大（>85kg）三种体型。
3. **品味系数（个性化偏好记忆）算法**：引入了用户自定义气囊状态与体压分布数据的绑定记忆功能。二次入座时，系统能够基于历史自定义参数基准进行自适应调节，实现“千人千面”。
4. **全闭环动态自适应调节**：实现了“感知-分析-调节-反馈”的全闭环控制。行车中乘员体位变化导致的压力改变，系统能在<200μs内响应并实时动态调节气囊。

### 4.4 有益效果
1. **极致舒适**：柔性纤维传感器如衣物般自然贴合，透气舒适，解决了传统薄膜传感器带来的异物感。
2. **高可靠性**：传感器能够承受20万次3MPa冲击，长期使用后自动回弹无需二次校准，满足车规级耐久性要求。
3. **精准调节**：多区域气囊（腰托、侧翼、前腿等）协同控制，结合实时体压反馈，确保座椅型面始终贴合人体曲线，有效降低长途驾驶疲劳。
4. **隐私保护**：完全基于物理体压数据进行体型识别和自适应调节，无需车内摄像头，彻底消除用户的隐私顾虑。

## 6. 具体实施方式
**实施例1：首次入座自适应调节**
乘员X（体重60kg，中等体型）首次落座。柔性纤维传感器以13Hz的频率采集体压数据。算法根据体压数据判定为中等体型，且本地无乘员X的自定义数据。系统控制腰托、腿托、座椅侧翼气囊自动充气，直至体压传感器反馈的压力分布数据落入中等体型的正常舒适阈值内。此时车机屏幕实时显示气囊充气过程和体压云图。

**实施例2：品味系数记忆与调用**
乘员X在实施例1的基础上，觉得腰部支撑不够，通过车机界面的“自定义气囊调节”功能，手动增加了腰托气囊的充气量。系统记录此时各气囊的状态以及对应的体压分布数据，并存入本地数据库的“中等体型-乘员X”档案中。当乘员X再次落座时，系统识别其体型后，自动调用该自定义参数，直接将气囊调节至乘员X偏好的状态。

**实施例3：行车过程动态调节**
乘员在行车过程中因疲劳微微弯腰改变了坐姿，导致腰部体压急剧减小，臀部前侧体压增大。传感器实时捕捉到这一变化，主控单元判断当前体压分布偏离了预设阈值，随即控制腰部气囊适度充气填补空隙，同时微调腿部气囊，使整体体压分布重新回到舒适阈值范围内。整个过程自动完成，无需人工干预。
